# WinBox Linux 4.3

MikroTik WinBox packaged for Linux.

## Supported packages

- Debian/Ubuntu:
  - winbox_4.3_amd64.deb
- Fedora/RHEL:
  - winbox-4.3-1.x86_64.rpm

## Features

- Installed to /opt/WinBox
- Desktop launcher included
- WinBox icon included
- Linux application menu integration

## Installation

### Ubuntu/Debian

```bash
sudo apt install ./winbox_4.3_amd64.deb
```

### Fedora/RHEL

#### DNF

```bash
sudo dnf install ./winbox-4.3-1.x86_64.rpm
```

#### RPM

```bash
sudo rpm -ivh winbox-4.3-1.x86_64.rpm
```
